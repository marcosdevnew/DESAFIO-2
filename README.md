# 🚗 Wide Coverage — Landing Page Responsiva para Mobilidade Urbana

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white) ![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-success?style=for-the-badge)

Projeto de uma **landing page** desenvolvido durante os desafios do **DevClub**, com foco em **HTML**, **CSS** e organização de layout.

A proposta do projeto é apresentar uma interface moderna para uma empresa fictícia de mobilidade urbana, com foco em uma experiência visual agradável e um design limpo, alinhado às tendências atuais de UI.
